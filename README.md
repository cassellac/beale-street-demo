# Beale Street Theater: Strategic Digital Demo

##  Objective
This repository hosts the high-fidelity proof-of-concept for the **Beale Street Theater** digital transformation. It serves as the visual anchor for the proposed **"Digital Divorce" strategy**, separating the venue's nightlife identity from its non-profit community roots to maximize revenue and market clarity.

##  Key Strategic Features
1.  **The "Neon Noir" Identity:** A high-contrast visual system designed to capture Route 66 tourists and nightlife seekers (`index.html`).
2.  **Commercial Engine:** A structured event grid with explicit genre/price tagging to resolve local SEO fragmentation (`events.html`).
3.  **"Box Office Buddy" Chatbot:** A friction-reducing tool hardcoded with specific venue logic (Parking, "Beale Street Brews" integration, Ticketing) to prioritize service over sales.
4.  **Strategic Asset Hub:** A centralized "War Room" housing key internal documentation, case studies, and brand guides (`marketing.html`).

##  Repository Structure
* `index.html` - **The Hook:** Homepage featuring the "Midnight Asphalt" aesthetic and Brand Promise.
* `events.html` - **The Engine:** Commercial interface demonstrating Tiered Pricing visibility.
* `marketing.html` - **The Brain:** Internal documentation hub utilizing "Center Peak" modals.
* `assets/` - Includes the custom "Route 66 Pattern" and Neon Brand Marks.

##  Tech Stack
* **Core:** Semantic HTML5
* **Styling:** Tailwind CSS (CDN) + Custom CSS Variables for Brand Systems
* **Typography:** Montserrat (Stability) & Playfair Display (Heritage)
* **Interactivity:** Vanilla JavaScript - Chatbot Logic & Modal State Management
